# Merge FBMs and meta files

A function to join by row individual FBMs and .info files into a unique
FBM and a unique .info file, merged_bm.

## Usage

``` r
merge_FBMs(filepath)
```

## Arguments

- filepath:

  Path to the individual FBMs.

## Value

A list with the merged FBM, the merged info file and the FBM header.
