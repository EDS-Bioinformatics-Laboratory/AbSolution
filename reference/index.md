# Package index

## Launch the app

Start the AbSolution Shiny application

- [`run_app()`](https://eds-bioinformatics-laboratory.github.io/AbSolution/reference/run_app.md)
  : Run the AbSolution Shiny Application Launches the interactive
  AbSolution Shiny application for exploring B-cell and T-cell receptor
  repertoire data. The application provides tools for loading
  AIRR-formatted datasets, inspecting sequence characteristics,
  visualizing clonotype distributions, and generating summary reports.
